# Silent Suzuka Fan Site - 沈黙の日曜日を越えて

伝説の名馬、そして『ウマ娘 プリティーダービー』でも愛される**サイレンススズカ**への情熱を込めたファンサイトです。

## 🌟 サイトの概要
このサイトは、サイレンススズカの輝かしい足跡を記録し、その魅力を共有するために作成した個人の趣味プロジェクトです。

## 🚀 主な機能
- **ホーム (Index):** サイトの入り口と最新情報。
- **プロフィール (Profile):** 詳細なプロフィールと血統、戦績の紹介。
- **ヒストリー (History):** サイレンススズカの激動の軌跡。
- **3Dシミュレーション (Simulation):** 圧倒的な逃げを再現した3Dレースシミュレーター。
- **メモリアル (Memorial/Memoirs):** 彼女の生きた証と記憶。
- **掲示板 (Board):** 交流のためのスペース（※管理用）。
- **グッズ (Goods):** 関連アイテムの紹介。

## 💻 技術スタック
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- Three.js (3Dシミュレーション部分)

## 🎨 デザインのこだわり
サイレンススズカのイメージカラーであるグリーンと、彼女の「静かなる闘志」を表現するモダンで落ち着いたデザインを採用しています。
